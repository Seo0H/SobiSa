## 💸 서비스 소개

내가 사고 싶은 그 물건, 정말 필요할까요? 소비사는 당신의 과소비를 걱정합니다.

고민하는 물건을 검색하고 그 물건을 사는 대신 할 수 있는 일을 알려드려요.
